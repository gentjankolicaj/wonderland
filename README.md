# wonderland #

Personal project around cryptography.

### What is alice for? ###

* Cryptographic module
* Contains implementations about: ciphers, cryptanalysis, encoders, hash

### What is garden for? ###

* Externalized resource module for resources character frequency etc...
* Resource origin is supposed to be different like local file_system, redis, mysql

### What is bouncy-castle for? ###
* Cryptographic module
* Wrapper/util implementations around bouncy-castle dependencies about: symmetric, asymmetric keys, keystore, hash...

### What is rabbit-hole for? ###
* Gui app using alice module
